# 5 Ranger Regiment - Training Documentation Repository

> Centralized training, doctrine, SOPs, and operational reference material for the 5 Ranger Regiment Arma 3 unit.

---

## Overview

This repository serves as the primary knowledge base for all training and operational documentation used by the 5 Ranger Regiment.

The purpose of this repository is to:

* Standardize training across the unit
* Maintain up-to-date SOPs and TTPs
* Provide recruits with structured learning resources
* Preserve operational knowledge and lessons learned
* Ensure all members have access to current doctrine

## Standard Operating Procedures

All personnel are expected to remain familiar with:

* Radio Protocols
* Chain of Command
* Rules of Engagement
* Medical Procedures
* Mission Planning Standards
* After Action Reporting

Failure to follow current SOPs may result in retraining or removal from operational duties.

---

## Document Standards

When creating or updating documentation:

1. Use clear and concise language.
2. Follow existing formatting standards.
3. Include revision dates.
4. Keep information accurate and current.
5. Submit changes through pull requests when applicable.

## Contributing

Authorized personnel may contribute updates to:

* Training material
* SOP revisions
* MOS qualification guides
* Operational references
* After Action Reports

All changes should be reviewed by Training Command or Unit Leadership before being merged.
